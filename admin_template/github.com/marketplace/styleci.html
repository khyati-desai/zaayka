More about StyleCI

Automatically fix the code style of your PHP, Python, JavaScript, JSX, Flow, TypeScript, CSS, SCSS, and Less. Enable, configure, and go!

Continuous integration with a difference. Packed with features you just can't live without, StyleCI provides checks for both your open and closed source repos.

By analyzing every push or pull request made on your repo, StyleCI ensures that your code is always written against the standards you want it to be, with fully automatic fixing available. We integrate with GitHub commit status too, to keep you informed.

On our Pro plan, you get space for 25 private repos (either personal and/or from organizations), unlimited analyses, optional slack notations, and free access for your whole team.
